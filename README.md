# EndBeaconMemoGenerator

一个专门用来拼合 EndBeacon 风格表情包的网页工具。  
从素材区选择最多 3 张图片，默认加入下方字段，再选透明底或黑底，就能直接下载成 PNG。

## 功能亮点

- **YELLOW / GREY 双样式切换**：同一组素材可快速切换成两套配色风格。
- **最多 3 张图左右拼接**：按点击顺序组合成一张完整表情包。
- **素材下方直接显示图片名**：选择时更直观，不容易点错。
- **默认加入下方字段**：字段位于图片外部、正下方，默认会生成如 **“上面危险需要”** 这样的内容，并且可以继续编辑。
- **字段可微调**：默认字号更大，并支持直接调整字段字号，还可用上移 / 下移按钮微调纵向位置。
- **透明底 / 黑底导出**：方便按不同使用场景保存不同版本。
- **一键复制预览图**：在支持的浏览器里可直接复制 PNG 到剪贴板。
- **下载文件名自动生成**：加入字段时优先使用字段文本命名；不加字段时使用图片名组合命名，且不会带上引号。
- **移动端更好用**：手机上会自动切换成更紧凑的操作与素材布局。

## 使用方式

1. 先切换当前想用的图片样式。
2. 在素材区点选 1 到 3 张图片。
3. 默认会插入下方字段，你也可以按需要关闭或修改字段文字。
4. 决定导出背景是透明底还是黑底。
5. 直接下载最终 PNG。

## 适合这样玩

- 拼一句自己的 EndBeacon 风格短句
- 做聊天、群聊、评论区里的即时反应图
- 用同一组素材快速导出不同配色、不同背景版本

## 一句话总结

**选图、拼图、改字段、下载。**  
把熟悉的素材快速组合成你想要的那张表情包。
